TL; DR Absent A's is a web application that assists students in studying for tests and quizzes using IBM's Watson.

I helped develop an app that uploads an audio file and utilizes IBM Watson’s APIs (speech to text and alchemy) to translate the speech in the file to text and then analyze the text and generate quiz questions based on the content of the file. The app was intended for students who might be absent from class but still needed to know the material covered. We submitted Absent A's for HackPSU 2016 and were a winner for the IBM challenge.
